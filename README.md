# A personal online document editor

`git clone https://github.com/duanjial/my-doc.git`

## Start the server

### First time run

```
cd server
npm i
```

### Other time run

```
cd server
npm run devStart
```

## Start the react frontend

### First time run

```
cd client
npm i
```

### Other time run

```
cd client
npm run start
```
